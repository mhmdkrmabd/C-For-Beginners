# 002- Drawing a Shape

Print a simple triangle:

```c
#include <stdio.h>
#include <stdlib.h>

int main()
{
    printf("      -- \n");
    printf("    /   \\ \n");
    printf("   /     \\ \n");
    printf("  /       \\ \n");
    printf(" /         \\ \n");
    printf("/           \\ \n");
    printf("-------------\n");

    return 0;
}
```
